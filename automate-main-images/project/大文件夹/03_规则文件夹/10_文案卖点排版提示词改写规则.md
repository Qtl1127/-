# 文案卖点排版提示词改写规则

## 目标

根据用户最新参考图，将太阳镜主图提示词改写为更强电商转化版式：品牌角标清晰、大标题冲击、卖点卡有层级、底部腰带明朗、场景明媚高级，同时严格保留太阳镜本体。

## 改写结构

每次改写太阳镜主图提示词时，必须包含以下模块：

1. 产品保真：锁定原图眼镜的镜框、镜片、鼻梁、镜腿、铰链、螺钉、比例、颜色、材质和反光效果。
2. 人物与场景：成年写实欧美男性模特，人物与前图无关联感；场景按产品气质独立选择。
3. 品牌角标：左上 `CKULH`，可做弧形角标、圆角品牌区、黑金运动角标、红金高光角标、深蓝金边角标或奶油金轻奢字标。
4. 主标题：左侧两行或三行强冲击中文大字，字重厚、层级强，可用黑金、白金、红金或深蓝金高光突出一个关键词。
5. 副标题或色带：标题下方加入短副标题、细线分隔或斜向色带，承接主卖点但不重复。
6. 图标卖点卡：左下 3-5 条，每条包含图标、主卖点和短补充说明；卡片必须有底板和边界。
7. 底部腰带：黑金、红金或深蓝金弧形功能栏，2-4 个图标分栏，金色描边或高光边清晰分离。
8. 负面限制：不得虚构认证、价格、销量、专利、医学功效、绝对化效果或原图不支持的功能。

## 版式风格

1. 品牌角标可参考红金弧形、黑金圆角、深蓝金边或极简白底品牌区，但不得遮挡大标题和产品。
2. 主标题应占左侧视觉核心，允许倾斜、厚重阴影、金属质感、高光描边、速度感斜切，但必须可读。
3. 标题可突出一个关键词，例如“强光”“视界”“偏光”“出行”“骑行”“变色”，但关键词必须与产品或提示词一致。
4. 左侧卖点卡不再只做简单白条，优先使用“圆形图标 + 金边卡片 + 主卖点 + 小字说明”的结构。
5. 底部腰带优先使用弧形黑金、红金或深蓝金横栏，形成电商主图收束感，不能和背景融为一体。
6. 腰带图标可使用眼睛、盾牌、太阳、羽毛、骑行、手提包、风线、镜片、山峰等线框图标，但不得暗示未支持功能。
7. 背景要明媚、有速度感或生命力，可使用海岸城市、山地公路、骑行道路、车窗阳光、夏日街景、度假街区等。

## 文案改写方法

1. 主标题公式：
   - 场景价值 + 佩戴价值：如“无惧烈日 / 型格出行”
   - 功能价值 + 场景价值：如“无惧强光 / 驾驶视界”
   - 气质价值 + 结果价值：如“锋芒视界 / 掌控格局”
   - 运动功能 + 视野价值：如“变色骑行 / 视界全开”
2. 副标题公式：
   - “一镜提升气场 轻松应对户外强光”
   - “为驾驶而生 为型格加分”
   - “自由探索 每一段风景”
   - “视野清晰 出行更从容”
3. 卖点卡公式：
   - 主卖点 4-6 字，补充说明 6-12 字。
   - 示例：`高清偏光 / 视野清晰 减少眩光`
   - 示例：`UV400 / 日常户外 安心佩戴`
   - 示例：`轻盈佩戴 / 久戴舒适 不压鼻`
   - 示例：`防眩视野 / 强光环境 更从容`
4. 底部腰带公式：
   - 2-4 个短卖点分栏，每栏 4-6 字主文案，可带 6-10 字说明。
   - 示例：`度假旅行 / 海边城市都适配`
   - 示例：`时尚穿搭 / 简约有型提升气质`
   - 示例：`清晰视野 / 日常出行更从容`

## 禁用和慎用文案

1. 不得使用没有依据的国家、产地、认证、销量、价格、专利、医学功效。
2. 不得使用“100%”“绝对”“第一”“唯一”“永久”“全天候保证”等绝对化表达。
3. “光感变色”“变色镜片”“日夜两用”只允许用于原图或提示词明确支持变色镜片的产品。
4. “骑行专用”“运动专用”只允许用于运动护目、骑行镜或提示词明确支持的产品。
5. “抗冲击”“TR90”“防滑稳固”“品质保障”“专家”等只允许在原图、提示词或用户明确提供依据时使用。
6. “修饰脸型”“百搭显气质”等偏风格表达可用于时尚款，但不得替代产品真实功能。

页面文字除 CKULH、UV、HD、POLARIZED、UV400 等专业术语外均使用中文。不得使用无依据的认证、价格、销量、专利、医学功效、国家产地、100% 绝对化承诺或原图不支持的功能。
