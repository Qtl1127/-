# CKULH 轻奢家居主图规则

## 适用条件

当任务、SPU 提示词或用户说明出现以下任意关键词时启用本规则：

1. CKULH
2. CKULH 轻奢家居类高转化主图
3. 1:1 方图电商主图
4. 左侧大文案 + 右侧产品场景 + 底部功能栏
5. 天猫/京东家居用品主图
6. 历史提示词中的 WCZ 轻奢家居美学或 WCZ 轻奢家居类高转化主图

本规则只适用于主图，不适用于详情页。无需读取竞品爆款图。

## 核心目标

基于商品原图生成 1 张 1:1 方图 PNG，风格为 CKULH 轻奢家居类高转化主图：商品真实、主体突出、信息清晰、商业感强。

## 必须保留

严格保留商品外形、结构、颜色、材质、比例、数量、配件、关键细节、商品自带包装文字和自带品牌标识。不得从零重造商品，不得把商品生成成另一种商品。

## 可以重做

可以重做原图背景、排版、营销文案、卖点标签、场景、光影、文案层级、功能标签、底部功能栏和局部高亮。重做内容不得改变商品本体或真实卖点。

## 品牌规范

1. 本批 36 个任务左上角品牌标识统一改为：`CKULH`。
2. 不再生成 `WCZ / 轻奢家居美学`，也不再把“轻奢家居美学”作为副文字。
3. LOGO 只作小型识别，建议占画面宽度 8%-12%，特殊构图不得超过 15%。
4. LOGO 底板可有可无；如使用，必须轻量、高级，并随产品风格微调。
5. 不同 SPU 不得机械复用同一角标、底栏、主色和构图模板。
6. 不得写错品牌名，不得出现乱码、重复字或错别字。

## 构图

采用电商视觉黄金比例：

```text
左侧或左上文案区 + 右侧或中右产品主视觉 + 轻量底部功能区
```

要求：

1. 商品放在右侧、中右侧或视觉黄金点附近，建议占主要视觉面积 55%-65%。
2. 左侧文案区放标题、副标题和少量核心信息，不堆叠卖点。
3. 底部功能栏必须轻量，不压产品；若承载卖点，应作为唯一卖点集中区。
4. 第一眼必须看懂商品用途，信息密度适中，不杂乱。

## 视觉与场景

1. 整体风格：轻奢家居、现代简约、干净高级、高转化、电商识别强。
2. 主色可按产品选择深绿、墨绿、青绿、米白、浅灰、黑色、香槟金等，不得固定套色。
3. 背景按商品真实用途选择厨房、浴室、收纳、清洁、桌面、墙面等家居场景。
4. 背景必须明亮、整洁、有生活感，不能抢商品主体或造成用途误判。

## 文案

1. 生成一个 6-12 字标题、一个 8-18 字副标题。
2. 全图合计 3-4 条关键卖点，宁少而准。
3. 标题、副标题、卖点卡和底部功能栏不得重复同一卖点。
4. 卖点只能来自商品品类、外观结构、使用场景、可见功能、提示词和旧图已有承诺。
5. 旧图已有的食品级 316、抗菌安全、密封防漏、次日达、赠品承诺等，可在真实存在时沿用或优化。
6. 不得虚构价格、销量、好评率、认证、专利、检测报告、承重数值、材质参数、规格参数、品牌背书、绝对化效果或未提供功能。
7. 不得使用“第一”“唯一”“顶级”“国家级”“100%保证”“永不”“绝对”“全网最高”“销量第一”等表达。

## 生图提示词核心

```text
请基于上传的商品原图，生成一张 1:1 方图 PNG 电商主图，风格为 CKULH 轻奢家居类高转化主图。

严格保留商品外形、结构、颜色、材质、比例、数量、配件和关键细节；不要保留原图背景、原图排版、原图营销文案和原图卖点标签。

画面采用“左侧或左上文案区 + 右侧或中右产品主视觉 + 轻量底部功能区”。左上角添加小型“CKULH”品牌标识，不再添加“WCZ / 轻奢家居美学”，LOGO 底板不得过大。

生成一个 6-12 字标题、一个 8-18 字副标题，全图只保留 3-4 条关键卖点，卖点不得重复。底部功能栏若承载卖点，则不要在其他区域重复。

配色、背景、光影和角标根据产品本身独立定制，避免与其他 SPU 套用同一模板。卖点必须基于原图、提示词和可见功能，不得新增无依据承诺或不可验证信息。

输出高清、真实、商业感强的专业天猫/京东家居用品主图，不出现错别字、乱码、重复文字或错误品牌名。
```

## 人工确认前展示

执行前展示：SPU、原图路径和数量、任务类型、商品品类判断、背景场景、拟标题、拟副标题、3-4 条关键卖点、底部功能栏用途、输出比例、输出格式、不可改变项摘要和生图任务说明。

## 质检与失败

生成后必须检查：

1. 是否 1:1 PNG。
2. 商品本体和自带文字/标识是否保持准确。
3. 是否添加小型 `CKULH`，且 LOGO 不过大。
4. 是否符合黄金比例构图，产品是否为第一主角。
5. 卖点是否 3-4 条、无重复、无冗余、无虚构。
6. 是否有错字、乱码、重复文字或错误品牌名。
7. 是否与其他 SPU 机械套用同一角标、色调、底栏或版式。
8. 光影、边缘、清晰度和商业主图感是否达标。

任一关键项不合格，按 `05_失败处理规则.md` 处理：不移动原图、不删除原图、不归档完成、记录失败原因、等待人工重新确认。
