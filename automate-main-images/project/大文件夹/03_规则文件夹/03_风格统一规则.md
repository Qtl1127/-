# 风格统一规则

## 适用范围

风格统一只作用于同一 SPU 内的多张图片。不同 SPU 不要求统一模板，反而必须避免机械套版。

## 同一 SPU

同一 SPU 多张图必须保持一致的视觉体系，包括色调、背景风格、光影质感、排版语言、字体气质、标签样式、文案层级、品牌调性和产品呈现方式。

提示词明确风格时优先遵守；提示词未写时，默认采用高级、干净、清晰、有质感、适合电商转化的统一风格。

## 不同 SPU

1. 不同 SPU 必须根据产品材质、颜色、用途、场景和价格感独立定调。
2. 不同单张任务同样按不同 SPU 管理，生成图片相互之间风格和色调不得一样，必须按照产品本身独树一帜。
3. 不得连续复用同一角标、底栏、主色、背景和版式。
4. 本次太阳镜批次同一品牌可以保持 `CKULH` 识别，但 `CKULH` 需要作为放大的主视觉品牌艺术字，按产品风格变化设计为深绿金边、极简黑字、奶油金、墨绿曲线角标等，不得做成固定模板。
5. 如果结果像“同一版式换产品”，判定为风格差异不足，需要重做或改提示词。
