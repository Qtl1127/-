# 不可改变项

## 产品本体

不得改变产品的外观、结构、形状、比例、材质、颜色、数量、配件、关键部件、包装文字和自带品牌标识。不得增加原图没有的核心结构，不得删除产品本体的重要结构，不得让产品变形、扭曲、融化、断裂或变成另一种商品。

## 太阳镜本体专项

本批太阳镜任务中，原图眼镜就是产品本体。生成目标是把这副眼镜准确佩戴到模特脸上，不能把它做成类似款。

1. 不得改变镜框形状、边角轮廓、上沿线条、镜圈厚度、鼻梁结构、镜腿粗细、镜腿长度、铰链连接位置和整体比例。
2. 不得改变镜片颜色、明暗、透光度、反光质感和材质效果。
3. 佩戴必须真实自然，左右对称，透视合理，镜腿自然挂在耳朵上，不漂浮、不扭曲、不穿模。
4. 不得加宽、变窄、变圆、变薄或改变眼镜材质质感。
5. 不得生成多余眼镜，不得遮挡眼镜主体。

## 卖点与承诺

不得虚构或夸大产品功能、参数、认证、奖项、检测报告、品牌背书、价格、销量、好评率、专利、承重数值和绝对化效果。原图或提示词没有依据的承诺一律不得新增。

已有卖点可以做更高级、更简洁的表达，但真实含义不能改变。

## 文字与品牌

1. 型号、规格、尺寸、容量、功率、颜色、材质等关键信息不得乱改。
2. 提示词提供文案时优先遵守；提示词为空时按规则生成。
3. 不得使用“第一”“唯一”“顶级”“国家级”“100%保证”“永不”“绝对”等高风险绝对化表达。
4. 不得改错品牌名，不得伪造其他品牌，不得添加未经要求的 Logo。

## 允许优化

在不改变产品和真实卖点的前提下，可以优化背景、光影、色调、排版、构图、层次、主体突出度、细节质感、场景氛围、文案视觉、标签样式和局部放大展示。
