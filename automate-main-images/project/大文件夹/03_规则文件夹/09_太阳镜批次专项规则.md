# 太阳镜批次专项规则

## 核心目标

本批图片围绕太阳镜产品本体进行 L3 电商主图升级。目标是把原图中的这副眼镜准确、真实、自然地佩戴到合规成年 AI 模特脸上，同时显著重构场景、构图、光影、版式和信息层级，形成高级、轻奢、有氛围感的京东天猫高转化太阳镜主图。

## 眼镜本体硬性要求

这副眼镜是产品本体，目标是准确佩戴到模特脸上，不得改变外形、框型、比例、镜腿、镜片颜色和材质，不能做成类似款。

1. 严格保留眼镜原貌，不改变镜框形状、边角轮廓、上沿线条、镜圈厚度、鼻梁结构、镜腿粗细、镜腿长度、铰链连接位置和整体比例。
2. 严格保留镜片效果，不改变镜片颜色、明暗、透光度和反光质感，不得换成别的镜片风格。
3. 佩戴效果必须真实自然，眼镜与脸部贴合正确，左右对称，透视合理，镜腿自然挂在耳朵上，不漂浮、不扭曲、不穿模。
4. 不得把这副眼镜变成其他类似款，不得擅自加宽、变窄、变圆、变薄或改变材质质感。
5. 模特角度优先采用正脸微侧或三分之二侧脸，以便准确展示镜框正面与部分镜腿结构。

## 人物质感要求

1. 人物必须为 25-35 岁左右的合规成年欧美男性写实模特。
2. 不要 AI 感假脸，不要过度磨皮，不要明星脸，不要夸张五官。
3. 脸部要干净真实，有商业广告摄影质感，皮肤保留自然纹理。
4. 不得出现未成年人、低俗姿态、性暗示、不合规裸露或手部畸形。
5. 如画面出现手部，手部必须自然、完整，不得遮挡眼镜主体。
6. 同批不同主图人物之间不得像同一人、同系列人物或存在关联关系；脸型、发型、肤色深浅、胡茬、年龄感、穿搭和气质需按产品独立变化。
