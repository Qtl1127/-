# 自动化生成主图

这是一个 Codex skill，用来批量把商品图片处理成 800x800 PNG 主图。

## 一句话安装

在 Codex 里复制下面这句话发送：

```text
$skill-installer install https://github.com/Qtl1127/-/tree/main/automate-main-images
```

安装完成后，重启 Codex，然后这样使用：

```text
$automate-main-images
```

## 注意

- 这个 skill 主要适合 Windows，因为里面的批处理脚本使用 PowerShell 和 System.Drawing。
